# KyrgyzPassportSBT Frontend

Минималистичный фронтенд для взаимодействия с контрактом KyrgyzPassportSBT в стиле портала Tunduk.

## Установка

1. Откройте `index.html` в браузере или используйте локальный сервер:
```bash
# Python
python -m http.server 8000

# Node.js
npx http-server
```

2. Откройте `http://localhost:8000` в браузере

## Настройка

Перед использованием необходимо:

1. Заменить `CONTRACT_ADDRESS` в `app.js` на адрес деплоя контракта
2. Обновить `explorerUrl` в функции `showSuccess()` на ваш блокчейн explorer (Etherscan, BSCScan и т.д.)

## Использование

1. Подключите MetaMask или другой Web3 кошелек
2. Введите ПИН паспорта (14 символов, например: ID123456789012)
3. Нажмите "Получить паспорт SBT"
4. Подтвердите транзакцию в кошельке

## Особенности

- Минималистичный дизайн в стиле Tunduk
- Автоматическая проверка статуса паспорта
- Валидация ПИН перед отправкой транзакции
- Обработка ошибок с понятными сообщениями
- Отображение tokenId после успешного минта
